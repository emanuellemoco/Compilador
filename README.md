## Para utilizar a calculadora bastar rodar:

```
$ python3 main.py 'expressão' 
```
Sendo 'expressão', a expressão que deseja calcular. 

###### Ex 1:
```
$ python3 main.py '3+5-1' 
```

######  Ex 2:
```
$ python3 main.py '7   + 2      - 1 '
```
